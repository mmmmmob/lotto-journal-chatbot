# Decision Log — Lotto Journal

Records all architectural decisions made in this project.
ADRs that are Accepted are source of truth for technical direction.

## Rules

- AI must read this index before making any architecture decision
- Never delete an ADR — change status to Deprecated or Superseded instead
- AI may create ADRs with status Proposed — humans approve

## Status Values

| Status     | Meaning                                    |
| ---------- | ------------------------------------------ |
| Proposed   | Suggested by AI or team; awaiting approval |
| Accepted   | Decided; use as the guiding direction      |
| Deprecated | No longer in use; history preserved        |
| Superseded | Replaced by another ADR                    |
