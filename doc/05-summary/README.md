# Summary Index — Lotto Journal

This folder stores monthly summaries and milestone retrospectives.

## Structure

```
05-summary/
  README.md          ← this file
  YYYY/
    YYYY-MM-summary.md       ← monthly summary
    MM/
      YYYY-MM-DD-summary.md  ← daily summary (local only)
    YYYY-MM-retrospective.md ← milestone or monthly retrospective
```

## Policy

- Monthly summaries: committed to git
- Daily summaries: local only (not committed)
- Retrospectives: written at the close of every milestone, or at least monthly

## Retrospective Template

See `core/13-retrospective-template.md` for the retrospective format.
Store retrospectives in `doc/05-summary/YYYY/`.

## Archive

Summaries are permanent — never archived or deleted.
